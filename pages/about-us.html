About Us

We have a mission: to dessiminate knowledge and respect for that knowledge. We want all who are interested to create and do more. Technology is our focus, especially as related to web technologies, programming, graphics, data wrangling.

Our Website

This website is created using:

  1. Python 3 - the language of choice for all-around getting things done.
  2. Pelican - a "static site generator", written in Python so that the resulting site can be run on a very simple host requiring few resources (no database, for example!).
  3. A custom theme called Cachet, which will soon be available on GitHub. There are many other themes currently available, so don't wait for this one to get started!

Photos and pictures are from unsplash and other places.
